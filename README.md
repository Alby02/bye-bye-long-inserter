# Bye Bye Long Inserter

This mod removes long-handed inserters from Factorio 2.0 and dynamically substitutes them with their raw manufacturing components across all recipes. It is highly recommended to use this mod alongside others that allow you to fine-tune standard inserter pickup and drop distances for greater flexibility.

## Features
- Removes base vanilla long-handed inserter from the game, hiding it from Factoriopedia and tech trees.
- Dynamically scans recipes requiring the long inserter and edits them to instead demand its base ingredients.
- Multiplies ingredient substitutions automatically based on recipe scaling requirements.
- Automatically isolates and strips removed inserters from Quality and Recycling logic.

## Mod Compatibility
Includes conditional settings to toggle removal of upgraded long inserters from:
- **Krastorio 2**
- **Factorio+**
